# Logo Component

Анимированный логотип компании Momentum Amplify с современными эффектами.

## Использование

```jsx
import Logo from "@/components/common/Logo";

// Базовое использование
<Logo />

// С кастомным размером
<Logo size="large" />

// Только иконка (без текста)
<Logo showText={false} />

// С дополнительными классами
<Logo className="my-custom-class" />
```

## Props

- **size** (string): Размер логотипа
  - `"small"` - маленький (иконка 8x8, текст text-lg)
  - `"default"` - обычный (иконка 10x10, текст text-xl)
  - `"large"` - большой (иконка 14x14, текст text-3xl)

- **showText** (boolean): Показывать ли текст "MOMENTUM AMPLIFY"
  - По умолчанию: `true`

- **className** (string): Дополнительные CSS классы

## Анимации

Компонент включает следующие анимации:
- Вращающийся градиентный фон
- Пульсирующее свечение
- Эффект искр (Sparkles)
- Плавный переход цветов на тексте
- Hover эффекты (масштабирование и вращение)
- 3D эффекты при взаимодействии

## Примеры

### В футере
```jsx
<Logo />
```

### В хедере / навигации
```jsx
<Logo size="default" />
```

### В Hero секции
```jsx
<Logo size="large" />
```

### Только иконка в мобильной версии
```jsx
<Logo size="small" showText={false} />
```

## Технические детали

- Использует Framer Motion для анимаций
- Градиенты: purple-600, blue-600, pink-600
- Иконки: Zap (основная), Sparkles (эффект)
- Полностью адаптивный дизайн
- Оптимизирован для производительности
